# UP250106-Python
### Proposito del proyecto
Este repositorio tiene la finalidad de almacenar mis trabajos realizados en la asignatura de Metodologia de la Programacion en la carrera de Ingeniería Mecatronica de la Universidad Politécnica de Aguascalientes, guardando actividades relacionadas con la clase y las indicaciones que nuestro supeiror nos proporcione. El alcance de este proyecto se centra en ayudarnos a mejorar nuestras habilidades en el area de programacion, adquiriendo habilidades utiles para la carrera en cuestión.

### Repositorio
Un repositorio en GitHub es un espacio en línea donde se almacena un proyecto (como código, documentos o archivos) junto con el historial de todos sus cambios, lo que permite saber qué se modificó, cuándo y por quién. Sirve para organizar, respaldar y compartir proyectos, facilitar el trabajo colaborativo, llevar control de versiones y mantener un registro ordenado del desarrollo, ya sea para uso personal, escolar o profesional.
